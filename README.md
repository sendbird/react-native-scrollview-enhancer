# react-native-scrollview-enhancer

ScrollView enhancer

## Installation

```sh
npm install react-native-scrollview-enhancer
```

## Usage

```js
import { ScrollviewEnhancerView } from "react-native-scrollview-enhancer";

// ...

<ScrollviewEnhancerView color="tomato" />
```
